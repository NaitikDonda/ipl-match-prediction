# IPL Match Predictor 🏏

ML-powered IPL match prediction web app with beautiful UI animations and stadium effects.

## 🚀 Live Demo
**https://ipl-match-prediction-v2.onrender.com/**

## ✨ Features
- 🎯 Pre-match winner prediction
- 📡 Live match win probability
- 🏟️ 40+ stadiums with auto city mapping
- 🎬 Cricket animations (bowling & batting)
- 🔊 IPL horn sound effects
- 🎨 Stadium background with lights
- 📱 Responsive design

## 🛠️ Tech Stack
- **Backend:** Flask, Python, scikit-learn
- **Frontend:** HTML, CSS, JavaScript
- **ML Models:** Random Forest, Gradient Boosting
- **Deployment:** Render

## 📝 Usage
1. Select teams and venue
2. Fill toss details
3. Click "Predict Winner"
4. Watch the animations and see results!

## 🏃 Run Locally
```bash
pip install -r requirements.txt
python api.py
```

Open `http://localhost:5000`

## 📊 Model
- Trained on IPL 2008-2025 data
- Features: team stats, venue, toss, city
- 75% live prediction accuracy
